### Что делает  сборщик ?
- Запускать  проект локально
- следит за изменением и пересобирает код
- обьединяет файлы для продакшена
- позволяет писать, подключает зависимости к import
- транспилирует код для старых браузеров
Vite - делает две важные вещи:
	1) Dev server - при запуске все изменения отображается в браузере мгновенно
#### Сборка на Продакшен
Vite на прод использует **Rollup.js**, а это значит код будет легким и оптимизирован
Под капотам разработки используется **Esbuild** 

#### В чем сила ?
- Быстрый старт
- минимум конфигурации
- горячая перезагрузка
- Плагины
- Поддержка и популярность
  
Vite - позволяет не тратить недели на настройку, а сразу начать кодинг и фокусироваться на продукте

- 2–3 часа в день (можно меньше, если без прокрастинации)
- Проект + теория + задачи
- Упор на то, что прокачивает на собесах

### 🥇 **Неделя 1 — “Боевые основы”**
- [ ] <span style="background:;border-radius:4px;padding:2px">День 1 </span>
#### ** Redux Toolkit + Zustand**

- [ ] <span style="background:;border-radius:4px;padding:2px">День 2 </span>
#### **День 2 – Git Pro + GitHub Flow**
- 📖 Почитать про `rebase`, `squash`, `cherry-pick`, `.gitignore`
- 💻 Практика:
    - Форкни open-source проект
    - Создай `feature` ветку → `commit` → `squash` → `pull request`

**Цель:** понимать, как вести код в реальных командах.

#### **День 3 – Асинхронность в JS**

- 📖 Повтори `Promise`, `async/await`, `try/catch`, `Promise.all`
    
- 💻 Напиши: асинхронный фетч данных с задержкой и loading-состоянием
    

**Цель:** быть готовым к каверзным async вопросам.

---

#### **День 4 – Задачи на собес**

- ⚔️ Попрактикуйся на [Frontend Mentor](https://frontendmentor.io), [JSHero](https://www.jshero.net/)
    
- 💡 Темы: строки, массивы, DOM, замыкания
    

---

#### **День 5 – Мини-проект: Todo + Redux**

- 💻 Сделай:
    
    - Добавление / удаление
        
    - Фильтрация
        
    - Локальное хранилище
        
- ☕ Оформи красиво, как будто это продакшн
    

---

#### **Выходные – релакс с пользой**

- ☁️ Смотри:
    
    - “What is a Closure?” (MPJ, JSConf)
        
    - “Why Redux is still used?”
        
- Или поработай над своим **портфолио** (лого, анимация, языки)

### 🥈 **Неделя 2 — “Собес-гладиатор”**

#### **День 6 – Testing the waters**

#### **День 7 – API + TS + Types**

- 💻 Напиши:

    - типизируй `response`, `params`, `User`
        

---

#### **День 8 – Advanced TypeScript**

- 📖 Учишь:

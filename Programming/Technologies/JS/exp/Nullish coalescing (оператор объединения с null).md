## ❓ `??` — nullish coalescing (оператор объединения с null)

Ты знаешь `?:`, а вот `??` — это **чисто для значений по умолчанию**, когда **переменная `null` или `undefined`**, но НЕ 0, пустая строка или false.

### Пример:
```js
let nickname = '';
let displayName = nickname || 'Безымянный';
console.log(displayName); // 👉 'Безымянный' ❌ хотя nickname — пустая строка, а не null
```
## Разница:
- `||` заменит даже `""`, `0`, `false`
- `??` заменит **только `null` и `undefined`**

### 🚀 Вспомогательная шпаргалка:

| Оператор | Что делает ?                   | Когда использовать                       |     |
| -------- | ------------------------------ | ---------------------------------------- | --- |
| `        |                                | `                                        |     |
| &&       | Возьми первое falsy            | Если оба условия должны быть true        |     |
| ??       | Возьми, если не null/undefined | Когда важно сохранить `0`, `""`, `false` |     |
|          |                                |                                          |     |

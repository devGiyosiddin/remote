#js-exp 
🕰 `setTimeout
``
```js
setTimeout(() => {
  console.log("Один раз через 2 секунды");
}, 2000);

```
⏳ **Один раз** с задержкой.  
Полезен, когда нужно выполнить что-то **однократно** с паузой.
---
🔁 `setInterval
``
```js
setInterval(() => {
  console.log("Каждые 2 секунды");
}, 2000);
```
🔄 **Бесконечно повторяется** с заданным интервалом.  
Используется, когда нужно делать что-то **периодически** (таймер, автообновление и т.д.).

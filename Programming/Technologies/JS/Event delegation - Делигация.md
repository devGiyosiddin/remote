**Делегация событий в JavaScript** — это техника, при которой ты назначаешь **один обработчик события на родительский элемент**, а не на каждый потомок. Это удобно, когда у тебя много элементов, особенно если они динамически добавляются.`
``
```js
document.querySelector("#list").addEventListener("click", function (e) {
  if (e.target.tagName === "LI") {
    console.log("Нажали на пункт:", e.target.textContent);
  }
});
```
🔹 Вместо навешивания обработчика на каждый `<li>`, мы слушаем клики на общем контейнере — `<ul id="list">`.  
🔹 Это экономит память и работает даже для новых элементов.
---
### 🔥 Зачем это нужно?
- **Производительность** (меньше обработчиков)
- **Гибкость** (работает с динамически созданными элементами)
- **Чистота кода** (всё централизовано)
